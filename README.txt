Code by Chance Cardona
Uses python3
To launch type:
	roslaunch wall_follower wall_follower.py

and the launch file should take care of the rest.
it may take a minute to start, this is to prevent errors
and is expected.
The robot swings a little bit wide but definitely follows the wall adequately.
